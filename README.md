# 🧺 Laundry Express Bot


The **Laundry Express Bot** is a sleek, modern, and interactive chatbot designed to help customers estimate their laundry service costs in seconds. Built with a focus on user experience and visual excellence, it bridges the gap between service providers and customers through a conversational interface.

## ✨ Features
*   **🌍 Multi-Language Support**: Fully localized in **English** and **Albanian (Shqip)**.
*   **💬 Personalized Interaction**: Greets users by name and provides tailored pricing based on their specific needs.
*   **💰 Dynamic Price Estimation**:
    *   **Various Clothing Category**: From suits and shirts to winter coats.
    *   **Dresses & Bedding Category**: Covers delicate dresses, blankets, and specialized items like down duvets.
    *   **Smart Math**: Automatically handles fixed prices and price ranges (e.g., estimations for "600-800 lek").
*   **🛒 Shopping Cart Logic**: Users can add multiple items to their order and see a live total update.
*   **🔥 Firebase Integration**: (Optional) Ready to connect to Google Firebase for hosting and saving user estimates to a Firestore database.
*   **💎 Premium Design**: 
    *   Glassmorphism UI with elegant blur effects.
    *   Modern **Outfit** and **Inter** typography.
    *   Smooth micro-animations and responsive layouts for mobile and desktop.

## 🛠️ Tech Stack
*   **HTML5 / CSS3**: Vanilla implementation with custom design tokens.
*   **JavaScript (ES6+)**: Custom state management for chat flows and cart calculations.
*   **Firebase SDK**: Integration for data persistence and hosting.
*   **Google Fonts**: Integrated for a professional, high-end feel.
